# AI-Based-Automatic-Modulation-Recognition-for-DVB-S2X-Waveforms
AI-Based Automatic Modulation Recognition for DVB-S2X uses deep learning to identify modulation schemes like 16-APSK, 32-APSK, and 64-APSK from I/Q signal data. It provides fast, accurate classification, signal visualization, and real-time analysis for satellite communication and SDR applications.
